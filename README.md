## **💡 Top Business Insights**
* **Promotional Fatigue:** Discovered that 33% of discounted transactions only yielded a **$1.15 spend increase** over full-price sales, suggesting a need for a "Threshold Discount" strategy.
* **Seasonality:** Mapped a massive revenue surge in **January** and a dip in **October**, allowing for data-driven inventory planning.
* **Precision Forecasting:** Developed a Random Forest model that predicts transaction totals with an average error of only **$23.94**.
